# 🤖 ML Projects

A collection of Machine Learning projects built while learning and practicing Data Science, Machine Learning, and Deep Learning. This repository contains projects ranging from beginner to intermediate level, focusing on solving real-world problems using Python and popular ML libraries.

---

## 📌 About

This repository is part of my Machine Learning journey where I implement different algorithms, perform Exploratory Data Analysis (EDA), preprocess data, train models, evaluate performance, and visualize results.

The goal is not just to build projects but to understand the mathematics, intuition, and implementation behind every algorithm.

## ▶️ How to Run

Clone the repository

```bash
git clone https://github.com/mayankkumarlinghe-jpg/ml-projects.git
```

Move into the project directory

```bash
cd ml-projects
```

Install dependencies

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook

```bash
jupyter notebook
```
